##  Installation

### Recommended: Icarus Mod Manager
1. Download [Icarus Mod Manager](https://github.com/Jimk72/Icarus_Software)
2. Extract and run `IcarusModManager.exe`
3. Click **Download Mod** and select desired mods
4. Double-click mods to add to install list
5. Click **Install All Listed Mods**
6. Mod Manager handles merging and conflicts automatically

### Manual Installation
1. Navigate to: `Steam\steamapps\common\Icarus\Icarus\Content\Paks\mods`
2. Create `mods` folder if it doesn't exist
3. Place `.EXMODZ` or `.pak` files in folder
4. ‚ö†Ô∏è Multiple mods may conflict ‚Äî use Mod Manager for best results
